Detailed Description

A canvas for form field (components), form info, and display info.

Manages each component in an array. Stores attributes to form. Does not contain display info, each individual component does.

addField (   $fobject)

Add a new field to the From.

The html name of the field must be unique. Name is used for an unqiue key in form.

addHTMLText (   $text)

May be deprecated: Adds HTML to form, uses html field.

Adds HTML directly into form without creating extra components. Used to be used for the early days of Alpha. Now is unnecessary due to component design decisions.

deleteField (   $nm)

Delete a field in form.

Return true if field deleted successfully. False if field does not exist with given name.

getArrayValue ( )

Get all the field's default value in form as an array.

The key for the returned array is the unique html name from the fields. All values, except HtmlText is returned.

getButtonName ( )

Get name of the button.

Note: Not the display text, but the name in html.

getButtonValue ( )

Get value/display text of the button.

Note: Not name attribute in html. It is the text displayed in the button. Refer to Form::getButtonValue().

getFields ( )

Gets an array of Fields within Form.

Returns an array with key as Fields name and value as Fields class.

getForm ( )

Get display contents for form.

Only returns display contents, does not display. Client must echo out the content returned.

getId ( )

Get SQL Id variable name.

getIdValue ( )

Get SQL Id variable value.

Gets the unique id variable value for SQL. Not the name of the variable in SQL. Refer to Form::getID() for name.

getValue (   $nm)

Get the field's default value stored in form based on the name of the component.

The name of the form is created by the constructor of the component. Refer to: Fields::__construct().

setArrayValue (   $post)

Set default value for field in form based on array.

For single values, use Fields::setValue($nm, $val). Array should have name of field as key and default value for field as value in array.

setButtonName (   $name)

Set button value for html.

Does not set the display text of the button. Modifies the name of button in html. Use this for css attributes.

setButtonValue (   $button)

Set display button string value/text.

Shows the text in the button. Only the display text. This does not affect the html name of the button. Refer to Form::setButtonName($name).

setFormName (   $nm)

Set the form name.

Sets the name of the form in html. When access forms or css, use the value set here. Default set to 'form'.

setId (   $id)

Set SQL Id variable name.

Sets the unique id variable name for SQL. Not the value of the variable in SQL. Refer to Form::setIdValue($idValue) for value.

setIdValue (   $idValue)

Set SQL Id value for Form.

Used for the unique id variable value for SQL. Not the name of the variable in SQL. Refer to Form::setId($id) for name.
