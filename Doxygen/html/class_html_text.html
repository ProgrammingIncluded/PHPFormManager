Detailed Description

Contains html text to output in Form.

Raw html can be stored and will out put accordingly. A very simple, yet elegant class. The text will output based on the order in which this field is added to the form. Only class that Form will skip when returning array values. As such, HtmlText will not be saved into SQL. Essentially, will be non-existent to the form class. If using this for formatting or css, refer to custom div settings for individual fields prior to using this HtmlText.
